# Disagreement Protocol Drafting Skill

## Procedure

1. Start from reviewed clause issues.
2. Keep current wording exact where available.
3. Draft proposed wording as a replacement or addition.
4. Include a concise rationale.
5. Add fallback wording when the issue is negotiable.
6. Preserve legal-review flags and uncertainty.

## Drafting Rules

- Use clear contract language.
- Avoid argumentative prose inside proposed wording.
- Do not include hidden assumptions in clause text.
- Keep rationale separate from wording.
- Prefer minimal edits when the source clause is mostly acceptable.

## Output

Each row should be usable in a protocol table.

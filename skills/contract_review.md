# Contract Review Skill

## Procedure

1. Read the case context and source clauses.
2. Preserve exact clause references and excerpts.
3. Tag each observation as fact, assumption, interpretation or unknown.
4. Identify issues that are appropriate for a protocol of disagreements.
5. Mark whether human lawyer review is required.

## Issue Types

- ambiguous obligation;
- excessive liability;
- missing limitation;
- unfavorable payment term;
- unclear acceptance procedure;
- one-sided termination;
- weak confidentiality;
- IP ownership issue;
- personal data or compliance issue;
- dispute resolution issue;
- missing attachment or definition.

## Quality Bar

Every material claim must be tied to source text or explicitly marked as an
assumption.
